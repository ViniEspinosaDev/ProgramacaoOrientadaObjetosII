    Atividade 01 - Nota AC1 - Individual

    Enviar tarefa
    • Entrega Segunda-feira por 23:59
    • Pontos 10
    • Enviando uma caixa de entrada de texto

     

    Baseado no exemplo apresentado no dia 09/03 (Ver repositório do Git), desenvolva um projeto que:

    Enviar o endereço do seu repositório no git.

    Não alterar o repositório após a data de entrega.

     

    • Existe um modelo chamado Produto (int codigo, String nome, double valor, int estoque)
    • A camada Repository possui 10 produtos cadastrados em um HashMap. Essa camada tem 2 serviços:
      • getProdutoById(int id)
      • getProdutos()
    • A camada Service tem quatro serviços (métodos):
      • getProduto(int id): devolve o produto e se não existir devolve uma mensagem de erro que deve ser apresentado em uma view.
      • getProdutosEmEstoque(): Devolve somente os produtos em estoque.
      • getProdutosByValorAcima(double valor): Devolve todos os produtos acima do valor passado.
      • getProdutosByValorAbaixo(double valor): Devolve todos os produtos abaixo de um valor passado.
    • A camada de apresentação deve chamar todos os serviços disponíveis na camada de Serviço. Usar um view para cada um dos serviços.

     

    Exemplos:

    http://localhost:8080/produto/1  (Links para um site externo.)

    (Apresenta em uma view os dados de produto de código 1 se existir, caso contrário apresenta uma mensagem de erro).

    http://localhost:8080/produtosEmEstoque (Links para um site externo.)

    (Apresenta todos os produtos em estoque em uma view)

     

    http://localhost:8080/produtosValorAcima/20 (Links para um site externo.)

    (Apresenta em uma view os produtos acima do valor 20)

     

    http://localhost:8080/produtosValorAbaixo/20 (Links para um site externo.)

    (Apresenta em uma view os produtos abaixo do valor 20)

     

    Dica:

    Para listar todos os produtos use:

     

    <table>
        <tr th:each="produto: ${produtos}">
        <td th:text="${produto.id}" />
        <td th:text="${produto.nome}" />
        ....
    </tr>
    </table>
